# Sales Analysis: Penjualan Retail Sembako
Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis data transaksi penjualan sembako untuk memahami tren operasional dan perilaku pelanggan. Analisis dilakukan menggunakan fitur Pivot Tabel di Excel untuk meringkas data yang kompleks menjadi informasi yang mudah dibaca.
Catatan: Data yang digunakan adalah dummy data yang bersumber dari materi pelatihan Bootcamp Excel oleh KarirNex.

Objektif Analisis
1. Mengetahui tren penjualan berdasarkan kategori produk.
2. Mengidentifikasi kota (City) dengan total pendapatan tertinggi.
3. Menganalisis efektivitas platform penjualan (Toko Hijau, Orange, Biru, dll).
4. Melihat korelasi antara Customer Rating dengan jenis kelamin pelanggan.

Tools yang Digunakan
1. Microsoft Excel: Pembersihan data (Data Cleaning) dan pembuatan Pivot Tabel.
2. Pivot Charts: Visualisasi data untuk mendukung hasil analisis dari tabel.

Langkah-langkah Analisis
Data Cleaning: Memastikan format tanggal, harga (Unit Price), dan total penjualan sudah sesuai (Currency).

Pivot Table Creation:
1. Membuat ringkasan Total Sales per Product Name.
2. Membuat perbandingan performa antar Platform.
3. Melakukan segmentasi pelanggan berdasarkan Jenis Kelamin dan City.
4. Visualization: Menambahkan Slicer (Filter) untuk memudahkan navigasi data berdasarkan waktu atau kota.
